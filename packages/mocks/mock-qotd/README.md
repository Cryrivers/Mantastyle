# @manta-style/mock-qotd

## Installation

```sh
$ npm install --save-dev @manta-style/mock-qotd
```

## Usage

```js
/**
 * @qotd
 */
value: string;
```

- For string only
- Generate random quote of the day from [API](https://talaikis.com/random_quotes_api/)
- A demostration of asynchronous mock plugins
