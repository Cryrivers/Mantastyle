# @manta-style/mock-example

## Installation

```sh
$ npm install --save-dev @manta-style/mock-example
```

## Usage

```js
/**
 * @example tanhauhau
 * @example cryrivers
 * @example jennieji
 */
userName: string;
/**
 * @example 5
 * @example 10
 * @precision 1
 */
score: number;
```

- Will generate data based on example, muliple examples will be chosen randomly.
